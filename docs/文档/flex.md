## 弹性布局方向

| class          | style                           |
| :------------- | :------------------------------ |
| flex-dir-col   | flex-direction: col;            |
| flex-dir-row   | flex-direction: row;            |
| flex-dir-rowRe | flex-direction: row-reverse;    |
| flex-dir-colRe | flex-direction: column-reverse; |

### 自定义

| 类型 | 参数             |
| :--- | :--------------- |
| 开关 | $flex-switch     |
| 前缀 | $flex-dir-prefix |
