TTcss

帮助开发者快速编写css代码的工具库

快速上手 →

高效开发

让开发者使用搭积木的样式编写css代码。

支持sass/less

同时支持sass和less。

可定制

支持自定义变量，更全面满足开发需要。

# 安装

# 安装
yarn add ttcss
# or
npm install ttcss
# or
pnpm install ttcss

# 引用

import 'ttcss/sass/index.scss'
